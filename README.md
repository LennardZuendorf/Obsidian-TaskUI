## Overview

**Shards TaskUI** is a plugin designed to enhance your task management experience in Obsidian by extending the functionality of the popular Tasks plugin. It helps users visually manage and break down tasks into manageable "shards," providing a clean and organized interface for task tracking.

## Features

- Integrates seamlessly with the **Tasks** plugin
- Allows users to visualize tasks in smaller, manageable pieces (shards)
- Desktop-only support

## Installation (Manually)

As this plugin is not yet available in the Obsidian community plugin browser, you will need to install it manually.
**Note**: This plugin requires the **Tasks** plugin to be installed and enabled.

1. Download the plugin files, unzip them, and rename the folder to `shards-taskui` (if necessary).
2. Place the folder in your Obsidian vault's `.obsidian/plugins/shards-taskui` directory.
3. Open Obsidian and navigate to **Settings > Community Plugins**.
4. Enable the **Shards TaskUI** plugin.

## Usage

After enabling the plugin, you can start managing them through the visual task interface provided by Shards TaskUI.
